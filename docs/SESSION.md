- Склейка по subjectID (требование WB API при merge imtID)

## Handoff (2026-06-18) — таймаут ИИ / фоновая задача

### Проблема

- ИИ-связки держали один HTTP-запрос на 10+ мин (много последовательных вызовов OpenAI, не WB)
- Render/прокси обрывал соединение → «долгое ожидание», UI мог зависнуть под overlay «коровы»

### Сделано

- POST `ai-suggest` → `{ task_id }`, работа в фоне, опрос `/api/tasks/{id}` с прогрессом
- Лимит `max_ai_batches` (по умолчанию 12), приоритет крупных категорий
- Настройка «Макс. запросов к ИИ за запуск» в UI
- Снятие overlay при ошибке/старте ИИ
- `app.js?v=52`

### Следующий шаг

Деплой → обновить страницу → загрузить каталог → ИИ → смотреть прогресс «запрос N/M»
