Настройки синхронизации

Правила применяются при ручном запуске и автозапуске для выбранного магазина Ozon.

«По порогу» — снимает товары со скидкой выше порога и добавляет подходящие. Legacy — только удаление из автоакций.

≤ порога — оставить или добавить в акцию; > порога — снять. SKU с нулевыми или странными ценами пропускаются.

Снимать товары с высокой скидкой

Удалять из акций товары, у которых скидка превышает порог.

Добавлять подходящие кандидаты

Автоматически включать в акции товары со скидкой не выше порога.

Исключить ваучерные акции

Не трогать промо с ваучерами и подарочными сертификатами.
