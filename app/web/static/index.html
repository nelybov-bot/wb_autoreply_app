Telegram

Один бот — все уведомления и отчёты. Пустой «отдельный» чат = используется основной ID чата.

Бот и чаты

Только строка от @BotFather, без «Bot» и без ссылки api.telegram.org.
Используется, если не задан отдельный чат ниже.

AI-ассистент в Telegram

Пишите боту в указанном чате — те же команды, что во вкладке AI: статистика, очередь, загрузка, генерация, автозапуск. Команды: /help, /new.

Только этот пользователь может управлять ассистентом. Несколько ID через запятую. Узнать свой ID: команда /id боту. Если пусто — любой в разрешённом чате.
Только этот chat_id может управлять. Несколько ID через запятую. Бот должен быть добавлен в чат.

Что отправлять

Шаблоны мгновенных сообщений (HTML)

Все сообщения в Telegram с parse_mode=HTML. Теги: <b> жирный, <i> курсив, <blockquote> цитата. Переменные подставляются с экранированием.

{store_name}, {product_title}, {source_label}, {customer_text}, {error_kind}, {explanation}
{telegram_title}, {store_name}, {deadline}, {consequence}, {summary} в <blockquote>, {action_needed}…

Проверка ошибок в карточке

При генерации ответа (вручную и в автозапуске) ИИ одним запросом формирует ответ и проверяет, нет ли в тексте покупателя признаков ошибки в карточке товара. Уведомления в Telegram — в разделе .

Важные уведомления Ozon

Сканирует чаты «Поддержка и новости» Ozon (штрафы, ИС, блокировки). ИИ анализирует сообщения с выбранной даты. Отправка в Telegram — в разделе .
