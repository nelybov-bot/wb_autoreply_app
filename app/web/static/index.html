Акции Ozon

Ozon сам добавляет товары в часть акций (автоакции — в API это поле auto_add_dates). Здесь можно посмотреть акции и удалить ваши товары из выбранных или из всех автоакций с участием.

Если пусто — автоудаление берёт все автоакции, где есть ваши товары. Если указаны ID — только эти акции (при ручном «Автоудаление сейчас» и в слоте автозапуска).
Выберите магазин Ozon и нажмите «Загрузить акции».

Автозапуск

Цикл: Загрузить новые → Сгенерировать → Отправить. За один слот берётся один магазин из отмеченных ниже (по очереди), чтобы не дергать API всех сразу. Чаты WB в слоте — только для этого же магазина (если это WB и есть ключ). Планировщик живёт внутри запущенного веб-сервера: на Render при «засыпании» инстанса слот по времени может не сработать — нужен постоянно работающий процесс или режим «по интервалу» после пробуждения.
Можно несколько, через запятую. Формат строго ЧЧ:ММ (например 09:00), иначе слот не сохранится.
Чаты: автоответ, если последнее сообщение от покупателя (до 5 чатов за слот). Акции: удаление ваших товаров из автоакций Ozon для магазина слота (настройки — вкладка «Акции Ozon»). Нужны Client-Id и Api-Key Ozon; для чатов ещё OpenAI.
Выбери конкретные магазины, без “все подряд”.
